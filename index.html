Projects & Artifacts

Monopoly Game Development

This project involved creating a digital version of Monopoly from scratch, utilizing Javalin for the backend with Java, and HTML, CSS, JS for the frontend. Despite being new to both Git and Javalin, our team managed to implement player movement and a scoring system within a one-month timeframe.

Reflection: This project was a significant learning curve, as it was our first time using Git and Javalin. Achieving what we did in just one month, especially when starting from zero knowledge on these tools, was a testament to our team's quick learning and adaptability. The experience taught us not only about the technologies but also about teamwork and the importance of clear communication in software development.

The Clean Divorce

A capstone project focused on building a website to support individuals through their divorce process, offering coaching and solutions to related challenges. Despite the limitations of using Wix, which constrained our coding capabilities, the project offered an extensive 8-month learning journey.

Challenges: Working within the confines of Wix presented unique challenges, particularly with custom coding limitations. However, these obstacles pushed us to explore creative solutions and leverage Wix's features to their fullest potential.

Reflection: Learning new technologies and navigating client expectations were pivotal aspects of this project. The culmination of our efforts was the client using our project as their main website, a testament to the practical impact and success of our work. Being a contributing member of this project's journey to production is a source of immense pride.

Books Search Website

This project entailed developing a website to search for books using the Google Books API, built with Vue.js, JavaScript, HTML, CSS, and SQLite. The complexity of Vue.js presented a steep learning curve, but mastering it proved invaluable for creating a dynamic, user-friendly interface.

Challenges: Integrating the Google Books API and navigating Vue.js’s intricate features were significant hurdles. The project demanded extensive research and self-study, particularly for understanding Vue's reactivity and component-based architecture.

Reflection: This project was a milestone in my development journey, showcasing my ability to self-learn and tackle complex APIs and frameworks. Achieving an 89% grade as a solo endeavor reflects not just on the project's success but also on my dedication to overcoming challenges and my capability in software development.
